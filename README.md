{Deep learning has become prominent in source code vulnerability detection because it can automatically learn complex feature representations from code without manually defining rules or patterns. However, many existing deep learning–based methods struggle to effectively leverage the syntactic structure and semantic dependencies of source code, and they often fail to account for data noise. As a result, these models show limited robustness when encountering complex or real-world code variations.To address this challenge, we proposes an approach called \textbf{D}ual-\textbf{c}hannel \textbf{M}ulti\textbf{m}odal \textbf{G}raph \textbf{N}eural \textbf{N}etwork\textbf{(DC-MMGNN)}. The model adopts a heterogeneous dual-channel architecture. In the syntactic channel, a hierarchical graph convolutional network (HGCN) with Jumping Knowledge connections is designed to extract multi-scale syntactic structural features from the Abstract Syntax Tree (AST). In the semantic channel, a Bayesian graph neural network is introduced to capture complex dependency relationships in the Program Dependency Graph (PDG) through parameter distribution modeling.To achieve optimal complementarity between heterogeneous features, we further propose an adaptive attention-based fusion mechanism that dynamically assigns modality weights according to the characteristics of each sample. In addition, the model integrates multi-scale parallel projection and hierarchical noise augmentation strategies during training to further improve generalization capability. Experiments conducted on public datasets such as VulCNN and QEMU demonstrate that the proposed method significantly outperforms several advanced baselines, achieving up to a 10.2\% improvement in F1-score.
